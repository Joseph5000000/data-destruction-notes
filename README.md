# data-destruction-notes (In accordance with CompTIA A+ material)
Degaussing (Specifically for wiping data of hard drives and other non-optical media drives electromagnetically)
Shredding (Shredding drives, particualry optical discs, into tiny pieces to reduce chances of data retrieval) 
Incineration (Method of using fire or a corrosive and flammable chemical to destroy drives, preferrably hard disk drives)
